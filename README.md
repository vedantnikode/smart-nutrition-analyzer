## Clone Repository

```bash
git clone https://github.com/vedantnikode/Smart-Nutrition-Analyzer.git

```

---

## Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend runs at

```
http://127.0.0.1:8000
```
